# 醫院手術排程系統優化 
* 解決問題：現階段醫院使用到的控台系統在畫面上不直覺，因此導致控台在管理刀房上不便利，因此本專案是為解決此問題而生。

* 使用工具：
  * 網頁製作：HTML、CSS、JavaScript
  * 後端框架：Node.js
  * 資料庫：MySQL
  
* 使用技術：Express + EJS + AJAX

* 目前功能：
  * 使用者登入，並在頁面上顯示使用者名稱
  * 各刀房讀取醫院現有刀房，為動態資料
  * 時間軸，用以對照現在時間
  * 從資料庫匯入所選取日期的手術排程
  * 點選手術跳出更詳細的資料
  * 系統每60秒讀取資料庫更新最新資料
  * 根據手術目前的狀態來給予對應的顏色
